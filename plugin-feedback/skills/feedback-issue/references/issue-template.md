# Issue テンプレート

Issue 本文は以下のフォーマットに従う。

## enhancement の場合

```markdown
## 対象プラグイン
{plugin-name}

## 説明
{改善要望の詳細}

## 期待する動作
{改善後のイメージ}

## 補足
{あれば}

---
_このIssueは plugin-feedback により作成されました_
```

## bug の場合

```markdown
## 対象プラグイン
{plugin-name}

## 説明
{バグの詳細}

## 再現手順
{わかれば}

## 期待する動作
{本来どうあるべきか}

## 現在の動作
{実際に起きていること}

## 補足
{エラーメッセージ等あれば}

---
_このIssueは plugin-feedback により作成されました_
```

## question の場合

```markdown
## 対象プラグイン
{plugin-name}

## 質問内容
{質問の詳細}

## 背景
{なぜその質問をしたのか}

---
_このIssueは plugin-feedback により作成されました_
```
